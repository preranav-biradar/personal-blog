# 🌐 Personal Blog Website

A modern and responsive **Personal Blog Website** developed using **WordPress**.  
This project focuses on creating a clean and structured platform to showcase blogs, projects, and personal information.

> ⚠️ **Note:** This project is currently running on a local environment. The GitHub repository is maintained for code versioning and demonstration purposes.

---

## 📌 Overview

The website is designed as a **content-driven platform** where users can explore blog posts, learn about the author, and view showcased projects. It follows a simple and intuitive layout to ensure a seamless user experience.

---

## 🧩 Pages & Structure

- **Home** – Introduction and landing section  
- **About** – Personal background and information  
- **Blog 1** – Detailed experience and key learnings from the *Avishkar Innovation Competition*  
- **Blog 2** – Achievement story on securing *First Position in the ENTC Department*, along with recognition and award   
- **Projects** – Showcase of personal and academic work  
- **Contact** – Communication page with contact form/email  

---

## ⚡ Features

- ✅ Built using **WordPress CMS**
- ✅ Clean and responsive UI design  
- ✅ Structured content layout  
- ✅ Easy navigation across pages  
- ✅ Customizable and scalable architecture
